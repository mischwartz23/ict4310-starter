Form

This is a web page with a form that will say hello to you if it works.

GET

This form submits a "GET" request to the server. All this does is ask for this page back again but puts the form parameter in the URL. Try entering a name in the box and clicking the "GET Request" button. Then look in the URL and you'll see it says "form.html?name=value" for whatever "value" you entered.

POST

This form submits a "POST" request to the server. It calls a different page that it just calls "form" and puts the parameters in the body of the request so that you don't see them.
